## Stack

A stack is a linear data structure that enforces a Last In First Out access pattern.

### Key methods are:
- push(T item) -- place on top
- T pop() -- remove top item
- T peek() -- return top value without removing it

Stack adapts an underlying structure (usually a List) for LIFO pattern.

Array and Link based implementations
- Array: use end
- List: use front
